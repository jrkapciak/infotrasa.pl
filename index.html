O projekcie

Infotrasa jest efektem pracy nad stworzeniem cyfrowego atlasu samochodowego wykorzystującego otwarte dane przestrzenne oraz wolne oprogramowanie. W tym celu wykorzystano bibliotekę języka JavaScript Leaflet, która znacząco ułatwia dostęp do informacji zgromadzonych przez użytkowników w serwisie OpeenStreetMap.

Serwis aktualnie umożliwia:

  •   wyznaczenie trasy przejazdu wraz z opisem trasy
  •  sprawdzenie aktualnych utrudnień na drogach zarządzanych przez GDDKiA
  •  lokalizację użytkownika
  •  sprawdzenie warunków atmosferycznych
  •  znajdowanie interesujących miejsc - na podstawie serwisu Wikipedia
  •  wydrukowanie mapy

  • W ramach projektu wykorzystano dane pochodzące z:
    API OpenWeatherMap, API OpenStreetMap, API OpenRouteService, GDDKiA , Wikipedia.
  • Oraz wtyczki do biblioteki Leaflet:
    Leaflet-Wikipedia , Leaflet-ContextMenu, Leaflet-EasyButton, Leaflet-easyPrint, Leaflet-LeafletRoutingMachine, Leaflet-Weather, Leaflet-Extra Markers, Leaflet.Locate.
